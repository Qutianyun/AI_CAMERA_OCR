This version does NOT run OCR in browser.
It receives OCR text from iPhone Shortcuts via URL parameter:
https://your-site/?ocr=<URL encoded OCR text>

Recommended shortcut:
1. Take Photo
2. Extract Text from Image
3. URL Encode extracted text
4. Open URL: https://qutianyun.github.io/AI_CAMERA_OCR/?v=12&ocr=<encoded text>
